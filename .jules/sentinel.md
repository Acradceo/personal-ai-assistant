## 2026-05-09 - Disable Flask Debug Mode\n**Vulnerability:** Flask application running with `debug=True` in production-like entrypoint.\n**Learning:** The Werkzeug debugger allows arbitrary code execution from the browser and should never be enabled unless strictly in local development environments.\n**Prevention:** Ensure `debug=False` (or omitted) in `app.run()` unless guarded by development environment checks.
